class mailing list

the 204421@cpe.ku.ac.th class mailing list is for questions and discussions related to this course. for example, questions related to the homework or topics you'd like to have more discussions as well as interested topics in computer networking that you found and would like to share with others in the class.  i will also join discussions and  use the list for class's announcements.

subscription: to subscribe to the list, send e-mail to the majordomo@cpe.ku.ac.th with the message subscribe 204421 in the body of the mail. the subject line can be left empty. 

join discussions: to ask questions or join discussions, send  your questions or comments to 204421@cpe.ku.ac.th. 

get off the list: if you want to get off the list, send e-mail to the majordomo@cpe.ku.ac.th with the message unsubscribe 204421 in the body of the mail.
