แนวทางการแก้ไขปัญหาการเชื่อมต่อระหว่างเมลเซิร์ฟเวอร์และเมลไคล์เอนต์ เช่น outlook ผ่านโปรโตคอล imap หลังจากการปรับปรุงระบบเมลเซิร์ฟเวอร์วันที่ 15 กรกฎาคม 2554 อาจส่งผลให้ผู้ใช้งานเมลไคล์เอนต์ผ่านโปรโตคอล imap ไม่สามารถเชื่อมต่อมายังเมลเซิร์ฟเวอร์ (nontri.ku.ac.th) ได้ โดยสามารถแก้ไขปัญหาได้ 2 วิธีคือ

1.      การปรับตั้งค่าคอนฟิกโปรแกรมเมลไคล์เอนต์

2.      ลบบัญชีอีเมลที่โปรแกรมเมลไคล์เอนต์แล้วสร้างใหม่

 

1.      การปรับตั้งค่าคอนฟิกโปรแกรมเมลไคล์เอนต์

ø microsoft outlook 2003/2007

1.       เมนู tools > account settings คลิกเลือกอีเมล แล้วคลิกที่ปุ่ม change…

2.       คลิกปุ่ม more settings…

3.       เลือกเมนู advanced แล้วที่ช่อง root folder path: ให้ลบ "inbox" ออก จากนั้นคลิกปุ่ม ok

 

ø microsoft outlook express 2003

1.       เมนู tools > account… คลิกเลือก nontri.ku.ac.th แล้วคลิกที่ปุ่ม properties

2.       คลิกปุ่ม more settings…

3.       เลือกเมนู imap แล้วที่ช่อง root folder path: ให้ลบ "inbox" ออก จากนั้นคลิกปุ่ม ok

ø mozilla thunderbird

1.       เมนู account settings คลิกเลือก server settings แล้วคลิกที่ปุ่ม advanced

2.       ที่ช่อง imap server directory: ให้ลบ "inbox" ออก จากนั้นคลิกปุ่ม ok

3.       คลิกขวาที่ inbox เลือก subscribe…

4.       คลิกเครื่องหมายถูกตาม folders ที่ต้องการ

5.       เลือก subscribe เสร็จแล้วคลิกปุ่ม ok

 

2.      ลบบัญชีอีเมลที่โปรแกรมเมลไคล์เอนต์แล้วสร้างใหม่

คู่มือการใช้งาน เข้าไปที่เว็บ nontri.ku.ac.th > manual
