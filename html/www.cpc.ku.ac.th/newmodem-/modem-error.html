1. ปัญหาเนื่องจากไม่ได้ต่อสายโมเด็มเข้าเครื่องคอมพิวเตอร์ จะปรากฏ error ดังรูปที่ 5.1


รูปที่ 5.1

เครื่องคอมพิวเตอร์ไม่พบโมเด็ม ให้ตรวจสอบการเชื่อมต่อโมเด็ม เช่น สายโมเด็มไม่ได้ต่อเข้ากับเครื่องคอมพิวเตอร์หรือสายหลวม ถ้าปกติลองเปิดสวิทช์โมเด็มและเปิดใหม่

2. ปรากฏ error ดังรูปที่ 5.2



รูปที่ 5.2

ในกรณีที่โทรศัพท์ของท่านมีระบบเสริมพิเศษ เช่น บริการฝากข้อความ ต้องทำการลบข้อความที่มีอยู่ใน voice mailbox ออกให้หมดก่อน จึงจะสามารถต่อโมเด็มได้

3. ปรากฏ error ดังรูปที่ 5.3



รูปที่ 5.3

เครื่องคอมพิวเตอร์สามารถติดต่อกับเครือข่ายได้แล้ว แต่อุปกรณ์เครือข่ายหรือเครื่องคอมพิวเตอร์ปลายทางไม่ตอบสัญญาณโมเด็มกลับมา อาจเกิดจากปัญหาของเครื่องคอมพิวเตอร์ต้นทางด้วย เช่น ค่าของ type of dialing-up server ไม่ได้กำหนดเป็น ppp: internet, windows nt server, windows 98

4. ปรากฏ error ดังรูปที่ 5.4



รูปที่ 5.4

สายปลายทางไม่ว่าง ให้ลองต่อใหม่ หรือเปลี่ยนหมายเลขโทรศัพท์หรือเบอร์โมเด็มเป็นหมายเลขอื่น

5. ปรากฏ error ดังรูปที่ 5.5



รูปที่ 5.5

ใส่บัญชีผู้ใช้ (user name) หรือ/และรหัสลับ(password) ไม่ถูกต้อง

คำแนะนำ
1. บัญชีผู้ใช้ (user name) และรหัสลับ(password) มีทั้งอักษรตัวใหญ่และตัวเล็ก ผู้ใช้ต้องพิมพ์ตามเอกสารที่ได้รับ
2. ตรวจสอบสถานะของปุ่ม caps lock
3. ตรวจสอบแป้นพิมพ์ให้อยู่ในโหมดภาษาอังกฤษ

6. ในกรณีที่โทรศัพท์ของท่านมีบริการเสริมพิเศษสายเรียกซ้อน เพื่อป้องกันการเชื่อมต่อโมเด็มเข้าสู่เครือข่ายนนทรีหลุดเมื่อมีโทรศัพท์เข้ามา

คำแนะนำ
ท่านควรจะยกเลิกการบริการสายเรียกซ้อนชั่วคราวในช่วงการการเชื่อมต่อโมเด็ม เช่นในกรณีที่ท่านใช้บริการสายเรียกซ้อนของ ta ควรยกเลิกโดยการกด #43#
และหลังจากการเชื่อมต่อโมเด็มเสร็จแล้ว ท่านสามารถเรียกใช้การบริการสายเรียกซ้อนโดยกด *43#
