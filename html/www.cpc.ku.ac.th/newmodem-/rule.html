เนื่องจากในปัจจุบันนี้มีผู้ใช้บริการบนเครือข่ายนนทรีเป็นจำนวนมาก จึงจำเป็นอย่างยิ่งที่ผู้ใช้งานทุกคนต้องปฏิบัติ ตามกฏระเบียบดังต่อไปนี้
1. ตั้ง password ที่มีความปลอดภัย และจะต้องไม่ลืม password ที่ตั้งขึ้นใหม่นี้
- password ที่ดีควรมีความยาว 6-8 ตัวอักษร
- ควรมีอักขระพิเศษรวมอยู่ด้วย เช่น $ & # % ! * + เป็นต้น
2. ไม่ยินยอมให้ผู้อื่นใช้งาน account และ password ของตนเองโดยเด็ดขาด
3. ไม่ทำการเปลี่ยน fullname ของ account จากที่ทางสำนักบริการคอมพิวเตอร์กำหนดให้
4. ไม่ออกมาใช้ shell อื่นนอกจาก restrict shell ที่กำหนดให้เท่านั้น
5. หากท่านทำการ subscribe ในการรับ message ทางเมลล์ ท่านจะต้องหมั่นทำการลบข้อมูลอยู่เสมอ
6. ไม่ทำการกระทำอย่างหนึ่งอย่างใดที่ส่อเจตนาจะทำลายระบบ
7. การใช้บริการ tin ไม่ควรมีการโต้ตอบในลักษณะที่ไม่มีสาระสำคัญ
8. ไม่ควรสร้างความรำคาญ หรือรบกวนการทำงานของผู้ใช้งานอื่นบนเครือข่าย
9. ห้ามล่วงเกิน หรือพยายามที่จะถอด password ของผู้อื่น
10. ห้ามส่ง mail แบบกระจายที่ไม่มีสาระสำคัญ
11. ไม่เก็บข้อมูลเกินกว่าที่โควต้ากำหนดใน home directory ของท่าน
12. ไม่ละเมิดสิทธิ์เข้าไปใน host ที่ตนเองไม่มีสิทธิ์ในการใช้งานโดยเด็ดขาด
13. การใช้งานบนเครือข่ายนนทรีนี้เพื่อประโยชน์ทางด้านการศึกษาเท่านั้น

หากผู้ใช้งานท่านใดฝ่าฝืนกฏระเบียบ จะได้รับการลงโทษโดยการยกเลิก account นั้นออกจากระบบ
