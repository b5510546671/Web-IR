ถาม    เป็นนิสิตได้รับ pincode แต่ลงทะเบียนรับรหัสผ่านใหม่จากเว็บไม่ได้ ทำอย่างไร ?
ตอบ    เป็นนิสิตได้รับ pincode แต่ลงทะเบียนรับรหัสผ่านใหม่จากเว็บไม่ได้ อาจมีสาเหตุมาจาก
ป้อนข้อมูลไม่ครบถ้วนทุกรายการ
ป้อนข้อมูลไม่ถูกต้อง เช่น รหัสประจำตัวนิสิต และ หมายเลข pincode

กรณีที่ป้อนข้อมูลครบทุกรายการ และกดปุ่มลงทะเบียน แล้วได้หน้าจอบอกรหัสผ่านใหม่ ให้รีบจดรหัสผ่านใหม่ทันทีและเก็บไว้เป็นความลับ ห้ามปรับแต่งหน้าต่างวินโดว์ก่อน เช่น ย่อหน้าต่าง ขยายหน้าต่าง เป็นต้น เพราะข้อมูลจะหายไป ไม่สามารถเรียกกลับมาดูอีกครั้งได้

กรณีที่ป้อนข้อมูลครบทุกรายการ และกดปุ่มลงทะเบียน แล้วได้หน้าจอแสดง server error แสดงว่าเครื่องเซิร์ฟเวอร์รับงานภาระหนักหรือมีข้อผิดพลาด ให้กลับไปป้อนข้อมูลใหม่

โฮมเพจการลงทะเบียนรับรหัสผ่านใหม่สำหรับเครื่อง nontri.ku.ac.th ให้บริการสำหรับเครื่องไคลเอนต์ที่อยู่ภายในเครือข่ายนนทรีเท่านั้น (158.108.x.x)


ถาม    ใช้รหัสผ่านใหม่แล้วไม่สามารถล๊อกอินได้ ทำอย่างไร ?
ตอบ    ใช้รหัสผ่านใหม่แล้วไม่สามารถล๊อกอินได้ ให้ลองตรวจสอบดังต่อไปนี้
1. ตัวอักษรเปรียบเทียบ
abcdefghijklmnopqrstuvwxyz0123456789!@#$%^&*?
abcdefghijklmnopqrstuvwxyz0123456789
2. ตรวจสอบตัวอักษรว่าเป็นตัวอักษรใด เพราะอักษรบางตัวมีความคล้ายกัน เช่น
- 1 (หนึ่ง) , i (ไอ เล็ก), l (แอล เล็ก), i (ไอ ใหญ่)
- 0 (ศูนย์), o (โอ เล็ก), o (โอ ใหญ่)

หากป้อนรหัสบัญชีและรหัสผ่านใหม่ถูกต้องแล้วล๊อกอินไม่ได้ ให้ลองใช้รหัสผ่านเดิมล่าสุดของผู้ใช้ก่อนวันที่ 18 ธ.ค.2543 เพราะข้อมูลของบางคนไม่สมบูรณ์จึงไม่ถูกเปลี่ยนรหัสผ่าน และถ้าเป็นผู้ใช้รายใหม่ที่ได้รับรหัสบัญชีและรหัสผ่านในระยะเวลาประมาณ 3 เดือนที่ผ่านมา ให้ลองใช้รหัสผ่านที่สำนักบริการคอมพิวเตอร์ออกให้

ถ้ารหัสผ่านเดิมและรหัสผ่านใหม่ใช้ไม่ได้ ให้ติดต่อฝ่ายเครือข่ายและสื่อสาร สำนักบริการคอมพิวเตอร์ หรือเขียนแบบฟอร์มขอรหัส password ใหม่ส่งมายังสำนักบริการคอมพิวเตอร์


ถาม    การปรับเปลี่ยนการเชื่อมต่อเข้าสู่เครือข่ายนนทรีทางโมเด็มจากระบบเก่าเป็นระบบใหม่ มีอะไรบ้าง?
ตอบ    การปรับเปลี่ยนการเชื่อมต่อเข้าสู่เครือข่ายนนทรีทางโมเด็มจากระบบเก่าเป็นระบบใหม่ มีดังนี้
1. ไอคอน dialup networking
1.1จากไอคอน dialup networking คลิ๊กเมาส์ขวาเลือกคำสั่ง properties ดังรูปที่ 1

รูปที่ 1 หน้าต่าง dialup networking

1.2 จะปรากฏหน้าต่างดังรูปที่ 2

รูปที่ 2 หน้าต่าง properties ของไอคอน dialup networking

1.3 ให้คลิ๊กที่ปุ่ม configure… เพื่อกำหนดคุณสมบัติของโมเด็มที่ใช้ในการเชื่อมต่อเข้าสู่เครือข่าย ดังรูปที่ 3

รูปที่ 3 หน้าต่าง properties ของไอคอน dialup networking

1.4 คลิ๊กที่แท็บ options ในส่วนของ connection control ถ้ามีเครื่องหมายถูกในช่องข้างหน้า bring up terminal window after dialing ให้คลิ๊กเอาเครื่องหมายถูกออกไป ดังรูปที่ 3

2. หน้าต่าง post terminal screen (หน้าต่างสีดำ)จะไม่ปรากฎในขั้นตอนการล๊อกอิน ซึ่งการล๊อกอินในระบบใหม่ทางโมเด็มจะกระทำครั้งเดียวทางหน้าต่าง connect to ดังรูปที่ 4


รูปที่ 4 หน้าต่าง connect to

3. การพิมพ์คำสั่ง slirp ppp ในการเปลี่ยนการเชื่อมต่อเข้าสู่เครือข่ายนนทรีทางโมเด็มด้วยระบบใหม่นี้ ผู้ใช้ไม่ต้องพิมพ์คำสั่ง slirp ppp


ถาม    เปิดอ่านจดหมายจากโปรแกรม ms outlook express หรือ netscape messenger แล้วปรากฏหน้าจอฟ้องว่ารหัสไม่ถูกต้อง ?
ตอบ    กรณีที่ผู้ใช้เปิดอ่านจดมายด้วยโปรแกรม ms outlook express หรือ netscape messenger แล้วกำหนดให้มีการจำรหัสผ่านไว้ด้วย ท่านต้องทำการเปลี่ยนรหัสผ่านบนโปรแกรม mail client ที่ใช้ให้ตรงกับรหัสผ่านบนเครื่อง mail server ก่อนจึงจะสามารถอ่านจดหมายได้ การตรวจสอบว่าท่านได้กำหนดให้มีการจำรหัสผ่าน ให้ตรวจสอบดังนี้
1. netscape messenger
1.1 หลังจากเปิดโปรแกรม netscape เสร็จแล้ว คลิ๊กที่เมนู edit
1.2 เลือกคำสั่ง preferences... จะปรากฏหน้าต่างดังรูปที่ 1

รูปที่ 1 หน้าต่าง preferences...

1.3 จากรูปที่ 1 หน้าต่าง preferences... เมนูด้านซ้ายมือให้คลิ๊กตรงเครื่องหมาย + หน้า mail & newsgroups แล้วเลือก mail server
1.4 จากหน้าต่างด้านขวามือเลือก incoming mail server ที่ใช้อยู่ เช่น ดังรูปที่ 1 เลือก incoming mail server ที่ชื่อ nontri.ku.ac.th แล้วคลิ๊กที่ปุ่ม edit
1.5 จะปรากฏหน้าต่างดังรูปที่ 2 ซึ่งมีเครื่องหมายถูกกำหนดให้มีการจำรหัสผ่านอยู่ (remember password)

รูปที่ 2 การกำหนดให้โปรแกรม netscape messenger ให้จำรหัสผ่าน

2. ms outlook express
2.1 หลังจากเปิดโปรแกรม netscape เสร็จแล้ว คลิ๊กที่เมนู tools
2.2 เลือกคำสั่ง accounts... จะปรากฏหน้าต่างดังรูปที่ 3

รูปที่ 3 หน้าต่าง internet accounts

2.3 จากหน้าต่างรูปที่ 3 ของ internet accounts ให้เลือกชื่อ account ที่ใช้งานอยู่ เช่น ดังรูปที่ 3 เลือกชื่อ account เป็น cpcsnm แล้วให้คลิ๊กที่ปุ่ม properties
2.4 จะปรากฏหน้าต่างดังรูปที่ 4 ให้คลิ๊กเลือกที่แท็บ servers จะเห็นว่ามีการคลิ๊กเรื่องหมายถูกให้จำรหัสผ่านอยู่ (remember password)

รูปที่ 4 การกำหนดให้โปรแกรม ms outlook express ให้จำรหัสผ่าน
