ใช้ในการติดต่อ (login) เข้าสู่อุปกรณ์เครือข่ายหรือเครื่องคอมพิวเตอร์ที่มีระบบปฎิบัติการเป็น unix หรือ linux เป็นต้น โดยใช้โปรโตคอลเทลเน็ตในการติดต่อ ซึ่งการใช้คำสั่ง telnet ผู้ใช้ (user) ไม่จำเป็นต้องนั่งตรงหน้าเครื่องคอมพิวเตอร์ (console) ถึงจะสามารถติดต่อเครื่องคอมพิวเตอร์นั้นได้ ผู้ใช้สามารถติดต่อ (login) จากที่ไหนก็ได้โดยที่ผู้ใช้จะต้องมีบัญชีผู้ใช้อยู่บนเครื่องนั้นก่อน
รูปแบบการเรียกใช้คำสั่งมีดังต่อไปนี้

telnet [ ชื่อเครื่อง(domain name)หรือหมายเลข ip-address]

โดยที่ผู้ใช้สามารถเรียกใช้ผ่านทาง ms-dos prompt ของ window หรือไปที่ start > run แล้วพิมพ์คำสั่ง telnet ชื่อเครื่องหรือหมายเลข ip address แล้วกดปุ่ม ok
หลังจากที่กดปุ่ม ok จะมีหน้าต่างถามชื่อบัญชีผู้ใช้(account name) และรหัสผ่าน(password) ผู้ใช้ต้องใส่ชื่อบัญชีผู้ใช้และรหัสผ่านของเครื่องที่ทำการติดต่อนั้น ขณะที่ผู้ใช้พิมพ์รหัสผ่าน บนหน้าต่างจะไม่ปรากฏตัวอักษรใดๆ ขึ้นมาทั้งนี้เพื่อความปลอดภัยและป้องกันไม่ให้คนอื่นเห็นรหัสผ่าน ซึ่งเมื่อระบบทำการตรวจสอบชื่อบัญชีผู้ใช้และรหัสผ่านว่าถูกต้อง จะแสดงเครื่องหมายเตรียมพร้อม (prompt) ขึ้นมา ในกรณีที่ผู้ใช้ใส่บัญชีผู้ใช้หรือรหัสผ่านไม่ถูกต้อง บนหน้าต่างจะขึ้นคำว่า login incorrect และจะให้ผู้ใช้ทำการ log in ใหม่อีกครั้งหนึ่ง
ยกตัวอย่างเช่น การติดต่อ (log in) กับเครื่องเชิร์ฟเวอร์นนทรีเพื่อทำการรับ-ส่งจดหมายอิเล็กทรอนิกส์ (e-mail)
