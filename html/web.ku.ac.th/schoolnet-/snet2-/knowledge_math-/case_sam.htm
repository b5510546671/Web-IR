   ตัวอย่าง   

จงเขียนกราฟของอสมการ    3x - 4y + 12 0

กราฟของเส้นตรง                3x - 4y + 12 = 0 ( เส้นทึบ )

จัดอสมการดังนี้ -4y     -3x  -  12
 y    3/4  +  3 

ดังนั้นกราฟของ y    3/4 x + 3 คือครึ่งระนาบของ 3x - 4y + 12 = 0 รวมถึงเส้นตรงด้วย

ในกรณีที่มีตัวแปรสองอสมการรวมกัน เราจะที่ว่าเป็นเงื่อนไขของการ and กัน ดังตังอย่าง

 4x + 3y    12
 x - y     0 

เขียนกราฟของอสมการทั้งสองได้
