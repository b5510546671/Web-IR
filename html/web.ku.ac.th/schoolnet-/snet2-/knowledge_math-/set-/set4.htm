ลักษณะของเชต ความสัมพันธ์ของเซต
สับเซต เพาเวอร์เซต
เอกภพสัมพัทธ์ แผนภาพของเวนน์-ออยเลอร์
ปฏิบัติการระหว่างเซต สัญลักษณ์ที่เกี่ยวข้องกับจำนวนต่าง ๆ ที่ควรทราบ


1. เซตที่เท่ากัน (equal sets) คือ เซตสองเซตจะเท่ากันก็ต่อเมื่อเซตทั้งสองมีสมาชิกเหมือนกัน
สัญลักษณ์ เซต a เท่ากับ เซต b แทนด้วย a = b
             เซต a ไม่เท่ากับ เซต b แทนด้วย a b

a = {1, 2, 3, 4, 5}
b = {1, 2, 3, 4, 3, 2, 5, 5, 5} 		เซต a มีสมาชิกเหมือนกับเซต b a = b
c = {a, e, i, o, u}
d = {i, o, u, e, o} 		เซต c มีสมาชิกเหมือนกับเซต d c = d
e = {0, 1, 3, 5}
f = {x | x i+, x < 6} 		เซต e มีสมาชิก 4 ตัว คือ 0, 1, 3, 5 แต่เซต f มีสมาชิก 5 ตัว คือ 1, 2, 3, 4, 5 e f
g = {สีแดง, สีน้ำเงิน, สีขาว}
h = {สีแดง, สีน้ำเงิน, สีเหลือง} 		สีขาว g แต่ สีขาว h g h

2. เซตที่เทียบเท่ากัน (equivalentl sets) คือ เซตที่มีจำนวนสมาชิกเท่ากัน และสมาชิกของเซตจับคู่กันได้พอดีแบบหนึ่งต่อหนึ่ง
สัญลักษณ์ เซต a เทียบเท่ากับ เซต b แทนด้วย a b

a = {a, b, c, d, e}
b = {1, 2, 3, 4, 5} 		a b แต่เซตทั้งสองมีจำนวนสมาชิกเท่ากัน และสามารถจับคู่แบบ 1:1 ได้พอดี
 a b
c = {x | x i+}
d = {x | x = 2n , n = 1, 2, 3, ...} 		c เป็นเซตจำนวนเต็มบวก {1, 2, 3, ...}
ส่วนเซต d เป็นเซตของจำนวนคู่ตั้งแต่ 2 ขึ้นไป {2, 4, 6, ...}
โดยสมาชิกของเซต c กับ d จับคู่แบบ 1:1 ได้พอดี
c d
หมายเหตุ 1. ถ้า a = b แล้ว a b
2. ถ้า a b แล้ว ไม่อาจสรุปได้ว่า a = b
