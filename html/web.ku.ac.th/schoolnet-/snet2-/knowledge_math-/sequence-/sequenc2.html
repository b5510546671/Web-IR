นิยาม

ลำดับ คือ ฟังก์ชันที่มีโดเมนเป็นเซตของจำนวนเต็มบวกตั้งแต่ 1 ถึง n หรือมีโดเมนเป็นเซตของจำนวนเต็มบวก  
       ลำดับที่มีโดเมนเป็นเซตของจำนวนเต็มบวก n จำนวนแรก เรียกว่า ลำดับจำกัด
             เขียนให้อยู่ในรูปของ พจน์ทั่วไป ได้ดังนี้:  
 
ลำดับ an = f(n)   เมื่อ n = 1, 2, 3, ... , n
     ลำดับที่มีโดเมนเป็นเซตของจำนวนเต็มบวก เรียกว่า ลำดับอนันต์
             เขียนให้อยู่ในรูป ของ พจน์ทั่วไป ได้ดังนี้:  
 
ลำดับ an = f(n)   เมื่อ n เป็นจำนวนเต็มบวก
