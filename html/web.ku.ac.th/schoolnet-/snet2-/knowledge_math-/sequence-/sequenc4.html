ตัวอย่าง

  ในช่วงวันหยุด 4 วัน นิภาได้วิ่งออกกำลังกายทุกเช้า โดยเริ่มวันแรกด้วยการวิ่งรอบสระน้ำ 3 รอบ และทุก ๆ วันถัดไปวิ่งเพิ่มขึ้นจากวันก่อนวันนั้น ๆ อีก 3 รอบ

เขียนตารางแสดงความสัมพันธ์ระหว่าง วันที่วิ่ง กับ จำนวนรอบที่วิ่ง ได้ดังนี้

ความสัมพันธ์ระหว่าง วันที่วิ่ง กับ จำนวนรอบที่วิ่ง เป็น ฟังก์ชัน ที่มี
โดเมน   =   {1, 2, 3, 4}
  เรนจ์     =  { 3, 6, 9, 12 }

เราเรียก 3, 6, 9, 12  ว่า  ลำดับจำกัด (finite sequence)
เพราะมีโดเมนเป็นเซตของจำนวนเต็มบวกเพียง n จำนวนแรก (n = 1, 2, 3, 4)
