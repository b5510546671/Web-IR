ช่วงของจำนวนจริง
(interval of real numbers)
เมื่อ a และ b เป็นจำนวนจริง และ a < b
    ช่วงเปิด
 (a, b) หมายถึง    {x | a < x < b}
    ช่วงปิด
 [a, b] หมายถึง    { x | a x b}
    ช่วงครึ่งเปิด
(a, b) หมายถึง    { x | a x < b}
    ช่วงครึ่งเปิด
(a, b) หมายถึง    { x | a < x b}
    ช่วง
(a, ) หมายถึง    { x | x > a }
    ช่วง
(a, ) หมายถึง    { x | x a }
    ช่วง
(-, a) หมายถึง    { x | x < a }
    ช่วง
(-, a) หมายถึง    { x | x a }
ช่วงครึ่งเปิด
(half-open interval)
ช่วงครึ่งเปิดของจำนวนจริง a, b เมื่อ a < b ใช้สัญลักษณ์ [ a, b) หรือ (a, b] โดยที่
      [a, b)   หมายถึง   { x | a x < b}
      (a, b]   หมายถึง   { x | a < x b}
ช่วงเปิด
(open interval)
ดูในเรื่อง ช่วงของจำนวนจริง
