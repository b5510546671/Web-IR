เพิ่มความเร็วในการเรียกใช้เมนู start

หากต้องใช้คำสั่ง regedit ในโปรแกรม windows 95 คุณควรใช้ความระมัดระวังอย่างยิ่ง หากการแก้ไขที่ทำลงไปเกิดข้อผิดพลาดก็จะก่อให้เกิดความเสียหายต่อการทำงานของ windows 95 ได้ บางครั้งอาจทำให้ถึงกับต้องติดตั้งโปรแกรมใหม่ เพราะฉะนั้นต้องระวังให้มาก

การเพิ่มความเร็วในการเรียกใช้เมนูเริ่ม ( start menu ) ที่เวบไซท์ http://members.prestige.net/~total/wintips/ ได้แนะนำให้ทำตามขั้นตอน ดังนี้

1. เรียกใช้คำสั่ง run ในเมนู start แล้วพิมพ์คำสั่ง regedit
2. เมื่อปรากฎไดอะล็อกบ็อกซ์ regedit เสร็จแล้วให้คลิก hkey_current_user / control panel / desktop
3. เรียกใช้เมนู edit คลิกคำสั่ง new เลือก string value
4. พิมพ์ menushowdelay ลงไป
5. คลิกที่ menushowdelay จากนั้นคลิกขวาเลือก modify แล้วกำหนดค่า delay ลงไป ( ตั้งแต่ 1 ถึง 10 โดยกำหนดให้เท่ากับ 1 จะเร็วที่สุด )
6. จากนั้นออกจากการทำงาน regedit และทำการ restart เครื่องขึ้นมาอีกครั้ง

ลองสังเกตุความเร็วในการเรียกใช้เมนู start เอาเองว่าเร็วขึ้นกันบ้างหรือเปล่า ถ้าหากเร็วไม่พอก็ให้คุณลองดาวน์โหลดโปรแกรมชื่อ microsoft powertoy utility tweak ui ที่เวบไซท์ของ microsoft มาลองปรับค่าการทำงานต่าง ๆ ภายใน windows 95 เผื่อจะช่วยให้ปรับการทำงานในแต่ละเมนูเร็วขึ้น
