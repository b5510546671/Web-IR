setup application
เนื่องจากว่า  ชุดโปรแกรมของ  foxpro  เป็นโปรแกรมขนาดใหญ่และมีให้เลือกใช้ทั้งระบบบนดอส  และบนวินโดว์  แต่สำหรับนักโปรแกรมมือใหม่ควรที่จะเลือกศึกษาชุดโปรแกรม foxpro  บนดอส  เพื่อศึกษาคำสั่งต่าง  ๆ  ให้เข้าใจให้มากที่สุด  และสำหรับชุดโปรแกรม foxpro  บนดอสนั้น  มีให้เลือกใช้กันหลาย  version  แต่  version  ที่นิยมใช้กันในปัจจุบัน คือ  version  2.6  for   dos  หรือที่รู้กันในนาม  sub-directory  fpd26  ซึ่งในการติดตั้งชุดโปรแกรมนั้น  หากเครื่องใดมีเครื่องอ่านซีดีรอมก็ทำได้โดยง่าย  โดยนำแผ่นซีดีรอมที่มีชุดโปรแกรม  foxpro  (หรือ  fpd26)   มาติดตั้งลงในเครื่องได้เลย  โดยโปรแกรมจะทำการติดตั้งไฟล์ที่ต้องใช้ในชุดโปรแกรมได้สมบูรณ์  ซึ่งโดยมากแล้วจะสร้าง  sub-directory fpd26  ขึ้นให้อัตโนมัติ  เป็นอันเสร็จสิ้นการติดตั้ง
