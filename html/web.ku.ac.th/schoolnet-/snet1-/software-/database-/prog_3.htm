how to use
ในการใช้งานชุดโปรแกรม  foxpro  นั้น  ผู้ใช้ต้องมีความรู้เกี่ยวกับโปรแกรมประเภทการจัดการฐานข้อมูลมาก่อน  อาทิ  dbase  หรือ  foxbase  เป็นต้น  หรือถ้าไม่มีความรู้มาก่อน ก็ต้องศึกษาไม่ว่าจะศึกษาด้วยตนเอง  หรือเรียนรู้จากสถานการศึกษาใด  ๆ  ก็ตาม  เนื่องจากว่าชุดโปรแกรม  foxpro  นั้น  มีใช้กันทั่วโลก  ต้นแบบของโปรแกรมจึงเป็นภาษาอังกฤษ 
ดังนั้นหากผู้ใช้ต้องการที่จะจัดเก็บข้อมูลเป็นภาษาไทย  จะต้องหาโปรแกรมประเภทภาษาไทย ที่เป็นแบบ  resident  program  มาติดตั้งก่อนที่จะใช้ชุดโปรแกรม  foxpro  โดยศึกษาจากชุดคำสั่งต่อไปนี้
 cd\fpd26
 c:\fpd26>thai
 c:\fpd26>fox
