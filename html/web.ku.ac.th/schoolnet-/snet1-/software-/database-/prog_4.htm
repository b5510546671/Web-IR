make  structure
ในการสร้างโครงสร้างไฟล์ฐานข้อมูลนั้น  เริ่มต้นด้วยการใช้คำสั่งในกรอบคำสั่ง  โดยพิมพ์คำสั่งต่อไปนี้
 create  <file  name>
ในช่องของ  <file  name>  เป็นการตั้งชื่อไฟล์ฐานข้อมูล  สมมติว่าตั้งชื่อไฟล์เป็น  student สามารถพิมพ์คำสั่งได้เป็น  create  student
หลังจากนั้น  สามารถที่จะกำหนดรายละเอียดเกี่ยวกับโครงสร้างไฟล์ฐานข้อมูลได้  อาทิ 
    • ชื่อของเขตข้อมูล  (field_name)
    • ประเภทของข้อมูล  (type)
    • ความกว้าง  (width)
    • จำนวนทศนิยม  เป็นต้น (decimal)
เมื่อกำหนดโครงสร้างไฟล์ฐานข้อมูลเสร็จแล้ว  สามารถบันทึกได้โดยการกดแป้นดังนี้
ctrl+w  หรือ  ctrl+end 
โครงสร้างไฟล์ฐานข้อมูล  จะถูกจัดเก็บไว้ในไฟล์  student และมีชนิดของไฟล์เป็น  dbf  อัตโนมัติ
