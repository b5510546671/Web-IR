index
จุดเด่นของโปรแกรมประเภทการจัดการฐานข้อมูลคือ  การเข้าถึงข้อมูลได้รวดเร็วโดยชุดโปรแกรม  foxpro  จะต้องกำหนดดัชนีก่อน  ด้วยคำสั่ง
  •  index  on  <expr>  to  <idx  file  name>
    โดยที่
      •   <expr>  เป็นนิพจน์ซึ่งประกอบไปด้วย  เขตข้อมูลเขตเดียว  หรือเป็นการรวมเขตข้อมูลหลาย  ๆ  เขต  โดยใช้เครื่องหมาย  +  ในการรวม
      •   <idx  file  name>  เป็นชื่อไฟล์ที่ต้องการตั้งชื่อให้กับไฟล์ดัชนี  ซึ่งจะมีชนิดของไฟล์เป็น  idx  อัตโนมัติ 
    เมื่อใช้งาน  จะต้องเปิดควบคู่กับไฟล์ฐานข้อมูล  สมมติว่าตั้งชื่อไฟล์ดัชนีเป็น  stdcode  สามารถใช้คำสั่งเป็น
  •  index  on  code  to  stdcode
    ในการใช้งานนั้น  จะต้องเปิดใช้งานควบคู่กับไฟล์ฐานข้อมูล  เช่น
  •  use   student  index  stdcode
    หรือสามารถใช้อีกรูปแบบ  คือ
  •  use  student
  •  set  index  to  stdcode
