1. เชื่อมโยงสาขาเพื่อการ online สำหรับสาขาที่ตั้งอยู่ในที่ห่างไกลมีระบบสื่อสารพื้นฐานไม่เพียงพอ
  2. เชื่อมโยงระบบสื่อสารจากสาขาเข้าศูนย์ระบบคอมพิวเตอร์โดยตรง
  3. ระบบสื่อสารผ่านดาวเทียมมีคุณสมบัติเฉพาะตัวที่สามารถส่งสัญญาณกระจาย ( broadcasting ) ไปถึง ผู้ใช้งานเป็นจำนวนมาก ๆ ได้ในเวลาเดียวกัน
  4. การประยุกต์ใช้ในระบบเงินฝาก ถอน โอน ที่สาขา
  5. การประยุกต์ใช้ในระบบเครื่องฝากถอนเงินอัตโนมัติ ( atm )
  6. การประยุกต์ใช้ในระบบบริหารงานสาขาและสำนักงานอัตโนมัติ
  7. การประยุกต์ใช้ในระบบตรวจสอบบัตรเครดิต ( credit card authorization ) เป็นต้น
