ปูนซีเมนต์ปอร์ตแลนด์

ปูนซีเมนต์ปอร์ตแลนด์ ถูกค้นพบโดยชาวโรมัน ด้วยการผสมขี้เถ้าภูเขาไฟและทรายกับน้ำ
พวกเขาพบว่า วัสดุก่อสร้างจะมีความแข็งแร็งกว่าที่เขาทำจากปูนขาวล้วน
ปูนซีเมนต์นี้ถูกเรียกว่า ปูนซีเมนต์พอซซูโลน่า ( pozzuloma  cement )
เพราะว่ามันถูกค้นพบครั้งแรกจากขี้เถ้าภูเขาไฟที่พอซซูโลน่า
ประเทศอิตาลี  ปูนซีเมนต์พอซซูโลน่า สังเคราะห์ทำจากกากโลหะ
