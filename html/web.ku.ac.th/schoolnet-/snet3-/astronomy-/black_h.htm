black  hole ( หลุมดำ )
 คือ    วัตถุที่สมมติมีความเร็วของการผละหนี ( escape  velocity )  ออกมามีค่าเท่ากับหรือความเร็วของแสง  ดังนั้นจึงไม่มีรังสีหรือวัตถุใดๆ  สามารถหนีออกจากมันได้แม้กระทั่งแสงและก็สามารถแผ่รังสีได้อย่างสมบูรณ์  เป็นผลจากการยุบตัวของดาวนิวตรอนที่ไม่สารถเอาชนะความโน้มถ่วงของตัวมันเองได้  เป็นผลทำให้มีการยุบตัวลงได้อีกจนดาวมีขนาดเล็กลง ๆ  จนในที่สุดอาจมีปริมาตรเป็นศูนย์และความโน้มถ่วงจะเพิ่มขึ้นเรื่อย ๆ
จนการเป็นอนันต์ ( infinity )  และจะพบว่ามีรังสีเอกซ์เรย์ที่แผ่ออกมาจากแบล็กโฮล  เนื่องจากการหมุนรอบตัวมันเอง   เมื่อก๊าซในบริเวณใกล้ตกลงไปในแบลกโฮลเพราะแรงดึงดูดมหาศาลดังนั้นจะไม่มีอะไรที่สามารถหลุดรอดออกจากหลุมดำได้  ซึ่งเป็นผลทำให้นักดาราศาสตร์ได้ค้นพบและศึกษา
